# Online News Website
## Using Tech
* PHP
* HTML
* CSS
* JavaScript
* Ajax
* JQuery
* Bootstrap
* MySQL
## Functionalities
* Displaying a list of news ordered by date, on home page.
* Displaying a list of news by category.
* Administrators can add, delete, or edit posts and categories.
